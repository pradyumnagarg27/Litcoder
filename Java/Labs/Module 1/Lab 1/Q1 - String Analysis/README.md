# Problem Statement
## String Analysis

To accomplish this, you need to break down the string and calculate the percentage for each category. The results are as follows:

Uppercase letters: 7.5%    
Lowercase letters: 65%    
Digits: 7.5%    
Other characters (symbols): 20%
